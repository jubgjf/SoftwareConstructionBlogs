# 软件构造：开发高质量的软件系统

## 课程目标

1. 认识软件构造的质量标准与目标，学习软件构造的基本过程
2. 学习 抽象数据类型ADT 和 面向对象编程 OOP
3. 关键质量目标
    - 可理解性
    - 可维护性
    - 可复用性
    - 健壮性
    - 时空性能
4. 代码重构等复杂高级构造技术
