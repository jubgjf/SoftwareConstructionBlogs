# 奇数阶幻方构造

P1 是一道关于幻方的题，通过这部分学习了一种奇数阶幻方构造法即 Siamese 算法 (opens new window)，算法流程如下：

  1. 把 111 放置在第一行的中间
  2. 顺序将 2,3,...2, 3, ...2,3,... 等数放在右上方格中
  3. 当右上方格出界的时候，则由另一边进入
  4. 当右上方格中已经填有数，则把数填入正下方的方格中
  5. 按照以上步骤直到填写完所有 N2N^{2}N2 个方格

由于幻方的对称性，也可以把右上改为右下、左上以及左下等方位

以一个 555 阶幻方为例：

# 向量偏转角问题

# 问题概述

P2 的 Problem 6: Calculating bearings 是一道有关向量顺时针偏转角的问题

给定一个向量的起始点 A，以及从 y 轴正方向顺时针旋转到这个向量的角度

再给定另一个点 B

求这个向量顺时针旋转到 B，需要旋转的角度

# 解决思路

由余弦定理，两个向量 a⃗\vec aa 和 b⃗\vec bb 之间的夹角 α(0≤α≤π)\alpha (0 \le \alpha \le \pi)α(0≤α≤π) 如下：

α=arccos⁡(a⃗⋅b⃗∣a⃗∣⋅∣b⃗∣) \Large \alpha = \arccos(\frac{\vec a \cdot \vec b}{|\vec a| \cdot |\vec b|}) α=arccos(∣a∣⋅∣b∣a⋅b​)

由于这个角在 000 和 π\piπ 之间，所以需要使用向量外积确定实际的角是在 000 和 π\piπ 之间还是 π\piπ 和 2π2\pi2π 之间

高中学的向量外积全忘了 😢 ，这里从头推导一遍

将平面向量拆成坐标形式

a⃗×b⃗=(axi⃗+ayj⃗)×(bxi⃗+byj⃗) \Large \vec a \times \vec b = (a_x \vec i + a_y \vec j) \times (b_x \vec i + b_y \vec j)a×b=(ax​i+ay​j​)×(bx​i+by​j​)

打开括号，得

a⃗×b⃗=axi⃗×bxi⃗+axi⃗×byj⃗+ayj⃗×bxi⃗+ayj⃗×byj⃗ \Large \vec a \times \vec b = a_x \vec i \times b_x \vec i + a_x \vec i \times b_y \vec j + a_y \vec j \times b_x \vec i + a_y \vec j \times b_y \vec j a×b=ax​i×bx​i+ax​i×by​j​+ay​j​×bx​i+ay​j​×by​j​

根据

i⃗×i⃗=0⃗j⃗×j⃗=0⃗i⃗×j⃗=k⃗j⃗×i⃗=−k⃗ \Large \vec i \times \vec i = \vec 0 \\ \vec j \times \vec j = \vec 0 \\ \vec i \times \vec j = \vec k \\ \vec j\times \vec i = - \vec k i×i=0 j​×j​=0 i×j​=k j​×i=−k

可得

a⃗×b⃗=(axby−aybx)×k⃗ \Large \vec a \times \vec b = (a_x b_y - a_y b_x) \times \vec k a×b=(ax​by​−ay​bx​)×k

所以当

axby−aybx<0 \Large a_x b_y - a_y b_x < 0 ax​by​−ay​bx​<0

时，是 a⃗\vec aa 顺时针转到 b⃗\vec bb，故

0<α<π \Large 0 < \alpha < \pi0<α<π

而当

axby−aybx>0 \Large a_x b_y - a_y b_x > 0 ax​by​−ay​bx​>0

时，是 a⃗\vec aa 逆时针转到 b⃗\vec bb，故

π<α<2π \Large \pi < \alpha < 2\pi π<α<2π

最后，当

axby−aybx=0 \Large a_x b_y - a_y b_x = 0 ax​by​−ay​bx​=0

时，a⃗\vec aa 和 b⃗\vec bb 共线，此时有

α=0 \Large \alpha = 0 α=0

或

α=π \Large \alpha = \pi α=π

而这两种情况是余弦定理能够区分的

# 凸包问题

P2 的 Problem 7: Convex hulls 是凸包问题，采用的思路如下：

  1. 点集如图所示

  1. 初始点是最左上方的点，初始方向向右

  1. 以当前的点为轴，依次计算从当前方向到其余各个点的顺时针偏转角

  1. 从所有偏转角中，取出偏转角最小的角对应的点

  1. 以这个点为轴，以此时的向量方向为基准，再次搜索偏转角最小的点

  1. 不断循环，直到回到初始点

  • -classpath 选项用于引入外部 jar 库，例如
